# Iris Flower Classification
## Project Overview

This project focuses on classifying iris flowers into three species — Setosa, Versicolor, and Virginica — using machine learning techniques. The classification is based on features like sepal length, sepal width, petal length, and petal width.

## Dataset

The dataset used is the famous Iris dataset, which contains:

150 samples
3 classes (species)
4 features:
Sepal Length
Sepal Width
Petal Length
Petal Width
